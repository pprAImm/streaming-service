# streaming-service

Модуль стриминга видео для платформы короткометражного ИИ-кинематографа.

## Назначение модуля
Этот сервис отвечает за высокопроизводительную, быструю и оптимизированную раздачу тяжелого видеоконтента (ИИ-фильмов и вертикальных клипов) конечным пользователям. Он минимизирует задержки при TikTok-скроллинге ленты.

## Основной функционал
* Оптимизированная раздача медиафайлов короткими чанками.
* Поддержка протоколов потокового вещания.
* Взаимодействие с объектным хранилищем (S3-совместимым), где физически хранятся сгенерированные видео.
* Кэширование популярных видеороликов для снижения нагрузки на инфраструктуру.

## Технологический стек
* **Язык разработки:** Golang
* **Окружение:** Проект инициализирован через Go Modules (`go.mod`)
